# Impeccable Design Context
# Comb Engine (comb-engine)

## Design Context

### Users
개인 사용자(lazybee). AI 멀티에이전트 파이프라인 제어 및 모니터링 인터페이스. 복잡한 AI 워크플로우 결과를 시각화하고 설정을 관리한다.

### Brand Personality
**AI 엔진 제어판** — 기술적인, 정밀한, 파워풀한. "복잡한 AI를 단순하게 제어".

### Aesthetic Direction
- LUDS 다크 테마 (globals.css 완전 적용 완료)
- 파이프라인 실행 상태 시각화 중심
- 메트릭/로그 중심 레이아웃
- glass-card 적극 활용 (이미 적용됨)
- **Anti-reference**: 일반 SaaS 대시보드 느낌

### Design Principles
1. **파이프라인 투명성** — 각 단계의 진행/성공/실패를 실시간 표시
2. **에러 우선** — 오류 상태는 항상 화면 상단에 노출
3. **결과 중심** — AI 출력 결과를 읽기 좋게 포맷
